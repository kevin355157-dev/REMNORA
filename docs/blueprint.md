# **App Name**: Cilantro Aroma

## Core Features:

- Dynamic Hero Section: Displays looping WebM video background, brand logo, '拾情話憶' and 'REMNORA' titles, a tea description, and two prominent call-to-action buttons. Includes right-side navigation with current item numbering (01, 02, 03...) and PREV/NEXT controls, with fade-in/fade-out text transitions and synchronized background updates.
- Comprehensive Loading Experience: Presents a full-screen loading animation on site entry, featuring the brand logo, a horizontal progress bar, and a percentage counter. The main Hero section is only revealed once all sequence images and assets are fully preloaded.
- Parallax Scroll Storytelling: Integrates visually immersive parallax scrolling effects throughout the content, specifically for the 'Brand Story' section, with animations synchronized directly to user scroll, ensuring full responsiveness across all devices.
- Full-fledged E-commerce System: Enables browsing of 10 tea products with detailed product pages, allows users to add items to a persistent shopping cart, manage quantities, and securely complete online payments for their orders.
- Intuitive Navigation & Theme Toggle: Provides a fixed navigation bar at the top of the page featuring the brand name 'CILANTRO', smooth-scrolling links to key content sections, and a dark/light mode switch for user preference.
- Interactive Content Sections: Structures various informational sections like 'Brand Story', 'Tea Introduction' (with notes on top, middle, and base notes), 'Ingredients & Benefits', an accordion-style FAQ, and a compelling Call to Action at the bottom.
- AI Tea Suggestion Tool: A tool that leverages AI to generate personalized tea recommendations or pairing suggestions (e.g., based on food, mood, time of day) for users, enhancing discovery and engagement.

## Style Guidelines:

- Primary color: #47663B (a deep forest green), conveying natural elegance and grounding, used for key textual elements and brand highlights.
- Background color: #FCF5D2 (a soft, warm cream), serving as the inviting and calm canvas for the entire website.
- Accent color: #E4F98F (a vibrant pale yellow-green), used to provide a fresh and energetic highlight for interactive elements and important details, creating a harmonious visual balance with the primary and background colors.
- Headline font: 'Alegreya' (serif), chosen for its elegant, intellectual, and contemporary feel, suitable for prominent titles. Body text font: 'Inter' (sans-serif), providing a modern, neutral, and highly readable experience for longer content blocks.
- Utilize minimalist, line-art style icons for social media links (Instagram, LINE, Facebook) in the footer, maintaining a clean and sophisticated aesthetic. The dark/light mode toggle should feature intuitive, recognizable symbols.
- A responsive, full-screen hero section is combined with a sticky top navigation bar. Content is organized into distinct, scrollable sections with clear visual hierarchy, adaptable to all device sizes, and incorporates a visually appealing two-column presentation for image-heavy sections where appropriate.
- Implement smooth fade-in/fade-out transitions for text elements in the Hero section and background sequence updates. Utilize subtle hover animations for interactive elements like CTA buttons and product cards, alongside scroll-driven parallax effects for depth.
